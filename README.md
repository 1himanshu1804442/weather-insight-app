# 🌤️ Weather Insight App

A dynamic, responsive web application that provides real-time weather updates using the OpenWeatherMap API. Built from scratch to showcase modern Vanilla JavaScript and asynchronous API integration.

## 🚀 Features
* **Real-Time Data Fetching:** Retrieves live weather conditions (temperature, humidity, wind speed) based on city search.
* **Dynamic UI Manipulation:** Automatically updates DOM elements and weather icons to reflect the current climate.
* **Modern JavaScript:** Implements ES6+ features including `async/await` and the `fetch` API for seamless, non-blocking data retrieval.
* **Responsive Design:** Clean and intuitive user interface built with HTML5 and CSS3.

## 🛠️ Tech Stack
* **Languages:** HTML, CSS, JavaScript (Vanilla)
* **API:** [OpenWeatherMap API](https://openweathermap.org/api)
